# Web Calculator
A simple calculator built for the web. It performs basic arithmetic operations and handles user input dynamically.<br>
This is project is part of the requirement of completeting the odin project foundation [Calculator](https://www.theodinproject.com/lessons/foundations-calculato)

## Features
It can perform basic math operations such as <br>
 * add 
 * subtract
 * multiply
 * divide

It features key buttons for entering digits, operators and clearing screen

_*The live Calculator could be accessed here [Calculator app]()*_
